# spheroid-to-spheroid-mc
Analytical MATLAB framework for spheroid-to-spheroid molecular communication
Spheroid-to-Spheroid Molecular Communication (MATLAB)
Overview

This repository implements an analytical framework for diffusion-based molecular communication between two spheroidal multicellular aggregates embedded in an unbounded fluid environment.

Both transmitter and receiver are modeled as porous media with reduced effective diffusivity compared to the surrounding medium.

Key Features
Analytical closed-form concentration solutions
Inside/outside spheroid diffusion modelling
Porous medium representation of multicellular aggregates
System-level performance evaluation
Validation against particle-based simulations
Model Applications
Organoid communication
Tissue engineering systems
Organ-on-chip platforms
Bio-inspired nanonetworks
How to Run

Open MATLAB and run:

main.m
Folder Structure
MATLAB/ → core analytical implementation
examples/ → simulation scenarios
validation/ → PBS comparison results
figures/ → generated plots
Validation

All results are validated using particle-based simulation (PBS).
